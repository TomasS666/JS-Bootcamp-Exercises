Eloquent JS Ch3 - Functions - E3: Bean Counting

Check the console! Press F12

You can call the function countBs and give it a string like the following example: countBs( 'Basics for Beginners' ) to check how many uppercase 'B' characters are in the given string.

Or you could call the countChar function which takes two arguments. The first argument is a string which you want to check, the second is a character which you want to know the amount of in the first given string.

Call it like this: countChar('Communicatie en Multimedia Design', 'e');
